aesdpy.servers¶

aesdpy.servers.get(api_url)¶

Get information about a AESD server.

Parameters:api_url (string) – The base url to query
Returns:The AESD server information
Return type:json

Example

import aesdpy
aesdpy.servers.get('http://1lv11lamb01.nrel.gov:8091/')
aesdpy.servers.get response¶
{
  "status": {
    "state": "ok",
    "message": "operating normally"
  },
  "server_id": "AESD Haystack",
  "server_type": "record_server",
  "models": [
    "RSF2v0"
  ],
  "version": 1
}

API Info:

GET /¶
>resjson string server_id:
 the server’s id
>resjson string server_type:
 the server type, e.g. “record_server”
>resjson string version:
 the AESD API version number
>resjson array models:
 a list of the server’s model ids
>resjson json status:
 the server’s status
aesdpy.servers.issue_command(api_url, body={})¶

Issue a server command.

Parameters:
  • api_url (str) – A url to query
  • body (json) – A command to issue
Returns:

The result of the issued command
Return type:

json

Examples

import aesdpy
body = {
    "command": "get_model_strategy"
}
aesdpy.servers.issue_command('http://1lv11lamb01.nrel.gov:8091/', body)

# Returns:
# {
#     "result" : "ok"
# }

API Info:

POST /command¶
reqjson string command:
 one of {“restart”, “clear”, “set_model_strategy”, “get_model_strategy”}
reqjson array(any) param:
 command parameters
resjson string result:
 the result of the issued command
Status Codes:
  • 200 OK – json
  • 400 Bad Request – error message
  • 500 Internal Server Error – error message
